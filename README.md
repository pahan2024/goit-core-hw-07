# Консольний бот-помічник (Адресна книга)

Цей проєкт є фінальною інтеграцією попередніх домашніх завдань у комплексний консольний додаток. Бот призначений для керування контактами, збереження телефонних номерів та дат народження користувачів, а також автоматичного розрахунку найближчих днів народження.

## 🚀 Основний функціонал

- **Керування контактами**: Додавання нових записів, оновлення існуючих, пошук номерів телефонів за ім'ям та перегляд усієї книги.
- **Робота з днями народження**: Можливість додавати дати народження до контактів у форматі `ДД.ММ.РРРР`.
- **Автоматичні нагадування**: Функція `birthdays` аналізує адресну книгу та виводить список людей, яких потрібно привітати протягом наступних 7 днів.
- **Перенесення вихідних**: Якщо день народження припадає на суботу чи неділю, система автоматично переносить дату привітання на найближчий робочий понеділок.
- **Валідація та безпека**: 
  - Номер телефону обов'язково має складатися рівно з 10 цифр.
  - Дата народження перевіряється на коректність формату.
  - Усі помилки введення обробляються за допомогою декоратора `@input_error`, що запобігає аварійному завершенню програми.

## 🛠️ Архітектура проєкту

Додаток побудовано на базі ООП із використанням наступних класів:
- `Field`: Базовий клас для полів запису.
- `Name`: Клас для зберігання імені контакту (обов'язкове поле).
- `Phone`: Клас для зберігання номера телефону з вбудованою валідацією (10 цифр).
- `Birthday`: Клас для зберігання дати народження з валідацією формату `strptime`.
- `Record`: Клас, який об'єднує ім'я, список телефонів та дату народження в один контакт.
- `AddressBook`: Клас, який наслідується від `UserDict` та відповідає за логіку збереження і пошуку всіх записів, а також за генерацію списку майбутніх святкувань.

## 💻 Список доступних команд

| Команда | Формат | Опис |
| :--- | :--- | :--- |
| `hello` | `hello` | Отримати стартове вітання від бота. |
| `add` | `add [ім'я] [телефон]` | Додати новий контакт або додати новий номер до існуючого контакту. |
| `change` | `change [ім'я] [старий_телефон] [новий_телефон]` | Змінити існуючий номер телефону для вказаного контакту. |
| `phone` | `phone [ім'я]` | Показати всі номери телефонів для вказаного контакту. |
| `all` | `all` | Вивести на екран усі контакти з адресної книги. |
| `add-birthday` | `add-birthday [ім'я] [дата]` | Додати дату народження у форматі `DD.MM.YYYY`. |
| `show-birthday`| `show-birthday [ім'я]` | Показати дату народження контакту. |
| `birthdays` | `birthdays` | Показати дні народження на найближчі 7 днів із перенесенням вихідних. |
| `close` / `exit`| `close` або `exit` | Коректно закрити програму та зберегти стан. |

## 📦 Як запустити додаток

1. Переконайтеся, що на вашому комп'ютері встановлено Python 3.8 або новішої версії.
2. Склонуйте цей репозиторій:
   ```bash
   git clone https://github.com/pahan2024/goit-core-hw-07
   ```
3. Перейдіть до папки проєкту:
   ```bash
   cd goit-core-hw-07
   ```
4. Запустіть головний скрипт програми:
   ```bash
   python bot.py
   ```
